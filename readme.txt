Redirect your url with the Pop Short Url.

== Description ==
The Main function of this plugin is to redirect your website urls to a pop up short url from ooorl.com

**More info:**

* [Shorturl](http://ooorl.com/)
* Find out [ooorl plugin](http://ooorl.com/plugin)

== Installation ==

1. Upload the contents of ooorl.zip to your plugins directory.
2. Activate the plugin
3. There will be a "ooorl" menu on your admin panel. 
4. Set the url you want to redirect.
6. That's all!
